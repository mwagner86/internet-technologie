Internet-Technologie

Eine Sammlung der Aufgaben von Maximilian Wagner

Aufgabe 1: Verbindung zum Webserver

Ein "hello world" HTML-Dokument per FTP auf einen Webserver laden, um die Grundlagen der Veröffentlichung zu erlernen.

Aufgabe 2: Webvisitenkarte in HTML

Erstellung einer persönlichen Webvisitenkarte unter alleiniger Verwendung von semantischem HTML, ohne CSS oder JavaScript.

Aufgabe 3: Webvisitenkarte mit CSS

Gestaltung der HTML-Webvisitenkarte mit reinem, handgeschriebenem CSS, um eine persönliche und optisch ansprechende Note zu verleihen.

Aufgabe 4: Responsive Webvisitenkarte

Umsetzung eines responsiven Designs für die Webvisitenkarte, das sich automatisch an verschiedene Bildschirmgrößen anpasst.

Aufgabe 5: Navigationselemente

Integration einer responsiven Navigationsleiste (Burger-Menü) unter Verwendung einer CSS-Library wie Bootstrap.

Aufgabe 6: Interaktivität mit JS Teil 1

Einführung von Interaktivität mit JavaScript, um das aktuelle Datum auszulesen und dynamisch auf der Seite anzuzeigen.

Aufgabe 7: Projektkonzept

Erstellung eines Konzepts im PDF-Format für die Prüfungsaufgabe, inklusive Zielsetzung, Umfang und Technologieeinsatz.

Aufgabe 8: Interaktivität mit JS Teil 2

Erweiterung der Interaktivität durch einen dynamischen Copyright-Hinweis, Event-Listener und eine Lightbox-Bibliothek.

Aufgabe 9: Wetterdaten-API

Einbindung aktueller Wetterdaten von einer externen API (OpenWeatherMap) mittels JavaScript und deren Darstellung auf der Seite.
